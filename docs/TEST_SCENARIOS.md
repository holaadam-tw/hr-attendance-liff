## 跨公司驗證 Checklist

### 測試身分

- 本米 admin
- 大正 admin
- platform admin

### 公司識別

- 本米：`fb1f6b5f-dcd5-4262-a7de-e7c357662639`
- 大正：`8a669e2c-7521-43e9-9300-5c004c57e9db`

### A. 本米 admin

| # | 情境 | 預期結果 |
|---|------|---------|
| 52 | 登入 `admin.html` | 只能看到本米公司名，沒有公司切換器 |
| 53 | 員工管理 | 只看到本米員工，看不到大正員工 |
| 54 | 請假 / 補卡 / 加班審核 | 只看到本米申請，看不到大正資料 |

### D. 必驗頁面

- `admin.html` 員工管理
- `requests.html` 或審核頁
- `schedule.html`
- `salary.html`
- `attendance_overview.html`
- 設定 / 公告 / 地點相關頁

### E. 失敗判定

出現任一項即視為跨公司隔離失敗：

- 本米 admin 看到大正員工姓名
- 大正 admin 看到本米請假或薪資
- 切公司後仍殘留上一家資料
- 公告 / 設定 / 地點改動影響另一家公司
- 匯出報表混入另一家公司資料
